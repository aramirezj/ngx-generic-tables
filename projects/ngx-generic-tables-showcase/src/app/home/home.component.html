ngx-generic-tables

What is:

The purpose of this library is to offer a wide selection of generic tools for forms powered by material through standalone components and directives

Requirements:

Angular material installed in your project

Installation:

The recomended version is Angular 15

To install it, you have to execute the command npm install ngx-generic-tables

To start using it, depends of the module you want to use, you have to import the module

Usage:

Depends of which utility are you are using, visit the sections using the side menu

Language support:

Currently available in english and spanish. The library will detect the locale through your {provide: LOCALE_ID, useValue: YOUR_LOCALE_ID } with the desired language
